**Smart Recipe & Fitness Service**  
*(2025.03.11 ~ 2025.03.31 — 21 days)*

SmartReciFit은 개인의 식습관과 영양 정보를 기반으로  
**맞춤형 레시피를 추천하는 요리 레시피 웹/모바일 서비스**입니다.  
사용자는 레시피를 검색·공유하고 후기를 남기며  
보다 건강한 식생활을 설계할 수 있습니다.

---

## 🌟 Main Features

### 🥗 개인 맞춤 레시피 추천
- 칼로리 및 영양소 기준 개인 맞춤 레시피 추천
- 다이어트·건강 관리 목적에 따른 식단 제공

### 📋 레시피 게시판
- 다양한 요리 레시피 검색 및 열람
- 즐겨찾기 기능 제공

### ⭐ 레시피 후기 & 커뮤니티
- 사용자 후기 작성 및 공유
- 댓글 및 좋아요 기능으로 소통

### 🎥 동영상 요약 레시피
- 요리 영상 핵심 요약 제공
- 빠른 레시피 학습 가능

### ⚖️ 정확한 재료 계량 정보
- 재료별 g / ml 단위 계량 정보 제공
- 초보자도 쉽게 요리 가능
